## Compile Contracts

```$ npx hardhat compile```
## Run Tests

```$ npx hardhat test```
## Deploy Contracts

```$ npx hardhat run scripts/deploy.js```
## Start Local Node

```$ npx hardhat node```
## Interactive Console

```$ npx hardhat console```
## Clean Project

```$ npx hardhat clean```
## Generate Gas Reports

```$ npx hardhat test --gas```
## Verify Contracts

```$ npx hardhat verify --network <network> <contract_address> <constructor_arguments>
```

`$ npx hardhat help`
